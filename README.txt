A complete solution to all the software needs of a restaurant.

The zip file in the google drive linked below contains all the files and a detailed documentation(much like in industry ) to the web app.
https://drive.google.com/file/d/1BS4axORXmIp3JFqVGLLdnQILwGjlagjg/view?usp=sharing


Software requirements:
- Python
- Anaconda
- Django 
- Djangorestframework
- Django-cors-headers

Instructions:
- Run the product in the command prompt
- Commands: python3 manage.py runserver5000
- Open the site 127.0.0.1:5000
